## 🚀 Getting Started

### **Step 1: Install Dependencies**
```bash
pip install -r requirements.txt
```

### **Step 2: Verify Installation**
```bash
python setup_verify.py
```

### **Step 3: Prepare Dataset**
Follow instructions in `TRAINING_GUIDE.md`:
- Download dataset (LJSpeech, LibriTTS, etc.)
- Organize as `dataset/wavs/` + `metadata.csv`
- Preprocess audio files

### **Step 4: Train Models**
```bash
# Train acoustic model
python training/train_acoustic.py \
    --dataset_path dataset/metadata.csv \
    --audio_dir dataset/wavs \
    --batch_size 32 \
    --epochs 1000 \
    --gpu 0

# Train vocoder
python training/train_vocoder.py \
    --dataset_path dataset/metadata.csv \
    --audio_dir dataset/wavs \
    --batch_size 16 \
    --epochs 500 \
    --gpu 0
```

### **Step 5: Run Inference**
```python
from inference.synthesizer import TTSSynthesizer

tts = TTSSynthesizer(
    acoustic_checkpoint='checkpoints/acoustic/acoustic_model_final.pt',
    vocoder_checkpoint='checkpoints/vocoder/generator_final.pt'
)

audio = tts.synthesize("Hello, world!", language='en')
tts.save_audio(audio, 'output.wav')
```

## 🎨 Example Usage

### **Basic Synthesis**
```python
from inference.synthesizer import TTSSynthesizer

tts = TTSSynthesizer(
    acoustic_checkpoint='checkpoints/acoustic/acoustic_model_final.pt',
    vocoder_checkpoint='checkpoints/vocoder/generator_final.pt'
)

# Simple synthesis
audio = tts.synthesize("Hello, world!", language='en')
tts.save_audio(audio, 'hello.wav')

# With prosody control
audio = tts.synthesize(
    "This is expressive speech!",
    language='en',
    pitch_scale=1.2,    # Higher pitch
    speed_scale=0.9,    # Slower
    energy_scale=1.3    # Louder
)
```

### **Voice Cloning**
```python
from inference.voice_cloner import VoiceCloner

cloner = VoiceCloner(
    acoustic_checkpoint='checkpoints/acoustic/acoustic_model_final.pt',
    vocoder_checkpoint='checkpoints/vocoder/generator_final.pt',
    speaker_encoder_checkpoint='checkpoints/speaker/encoder_final.pt'
)

# Clone voice
audio = cloner.clone_voice(
    text="This is in the cloned voice.",
    reference_audio='reference.wav',
    language='en'
)
```

### **Multilingual**
```python
# English
audio_en = tts.synthesize("Hello!", language='en')

# Spanish
audio_es = tts.synthesize("¡Hola!", language='es')

# Chinese
audio_zh = tts.synthesize("你好！", language='zh')
```
